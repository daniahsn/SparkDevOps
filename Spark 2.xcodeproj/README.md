## How to Pull and Run the Xcode Project

Our implementation prototype is stored in a GitHub repository and built using Xcode. Follow the steps below to clone and run the project locally.

### **Prerequisites**

* macOS with the latest version compatible with your Xcode installation
* Xcode installed from the Mac App Store

### **Clone the Repository**

Just clone the repo. 

### **Open the Project in Xcode**

1. Locate the `.xcodeproj` or `.xcworkspace` file inside the cloned repository.
2. Double-click to open it in Xcode, **or** open via Terminal:

   ```bash
   open *.xcodeproj
   ```

   or

   ```bash
   open *.xcworkspace
   ```

### **Build and Run**

1. In XCode select your iPhone or a simulator phone (preferably one of the newer ones)
3. Press the play button to build and run the application.
